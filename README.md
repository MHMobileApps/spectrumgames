## Run locally

```bash
python3 -m http.server 4173
```

Open <http://localhost:4173> and start from the hub.

## Wild Bunch controls

- `1-9`: menu options
- Any key (or `Space`) during duel after `HE MOVES!`
- `R`: restart from any screen
